# Neljansuora


#### Aihe: 

Toteutetaan kahden pelaajan peli, jossa ideana on samannimisen lautapelin mukaisesti pudottaa vuorotellen kiekkoja pelilautaan yrittäen saada aikaiseksi neljän kiekon suoran.

Pelilauta on 7 sarakkeen levyinen ja 6 rivin korkuinen. Molemmilla pelaajilla on käytössään eriväriset kiekot, joita pudotellaan lautaan vuorotellen tyhjiin sarakkeisiin. Voittaja ratkeaa, mikäli jompi kumpi pelaajista saa muodostettua laudalle neljän kiekon suoran joko vaakasuorassa, poikittain tai pystysuorassa. Mikäli lauta täyttyy ilman voittajaa, tulos on tasapeli.

#### Käyttäjät: Kaksi pelaajaa

Valikon toiminnot: 

* Peliä käynnistettäessä valitaan nimimerkit.

#### Pelin toiminnot:

* Pelaaja valitsee vuorollaan sarakkeen, jonne pudottaa oman kiekkonsa

* Peli havaitsee, mikäli jommalle kummalle pelaajalle muodostuu neljän suora, tai jos ruutu täyttyy ilman voittajaa.
